Jawab :

Keterkaitan antara SaaS, PaaS, dan IaaS dalam Cloud Computing

Dalam cloud computing, SaaS, PaaS, dan IaaS adalah tiga model layanan yang membentuk sebuah layered architecture atau tumpukan layanan yang saling bergantung. Ketiganya berperan menyediakan berbagai level abstraksi sehingga pengguna dapat memilih seberapa banyak kontrol dan pengelolaan yang ingin mereka lakukan.
1. IaaS (Infrastructure as a Service)

IaaS adalah lapisan paling dasar yang menyediakan infrastruktur TI seperti:
server virtual,
penyimpanan data,
jaringan,
dan resource komputasi lainnya.
Pengguna memiliki kontrol penuh terhadap sistem operasi dan aplikasi yang berjalan, tetapi tidak perlu mengelola perangkat keras fisik.
